STEP-BY-STEP SETUP (10 MINUTES)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 1: ENABLE VBA (1 minute)
──────────────────────────────────────────────────────────────────

1. Open Microsoft Excel
2. Go to File → Options
3. Click "Trust Center"
4. Click "Trust Center Settings..."
5. Click "Macro Settings" on left
6. Select "Enable all macros"
7. Click OK, OK

⏱ Time elapsed: 1 minute

STEP 2: CREATE WORKBOOK (2 minutes)
──────────────────────────────────────────────────────────────────

1. Create NEW blank Excel workbook
   File → New → Blank Workbook

2. Press Alt+F11 (opens VBA editor)

3. In VBA editor:
   File → Import File → Select "CreateERP_VBA.bas"

4. Press F5 to run macro

5. Workbook automatically builds all sheets and formatting!

6. Close VBA editor (Alt+Q)

⏱ Time elapsed: 3 minutes

STEP 3: IMPORT MAIN LOGIC (2 minutes)
──────────────────────────────────────────────────────────────────

1. Press Alt+F11 again

2. File → Import File → Select "RIAMAN_FASHION_ERP_Main.vbs"

3. Review the code (optional - it's well-commented)

4. Close VBA editor (Alt+Q)

⏱ Time elapsed: 5 minutes

STEP 4: SAVE WORKBOOK (1 minute)
──────────────────────────────────────────────────────────────────

1. File → Save As

2. Name: RIAMAN_FASHION_ERP.xlsm

3. File Type: Excel Macro-Enabled Workbook (*.xlsm)

4. Location: Documents\RIAMAN_FASHION_ERP\

5. Click Save

⏱ Time elapsed: 6 minutes

STEP 5: ADD SAMPLE PRODUCT (2 minutes)
──────────────────────────────────────────────────────────────────

1. Go to Products sheet (visible tab)

2. Scroll down to find "[+ Add New Product]" button

3. Click it and enter:
   Name:           Eternal Dreams Bridal Gown
   Category:       Wedding
   Size:           US 6
   Color:          White
   Fabric:         Silk Tulle
   Designer:       RIAMAN Couture
   Cost Price:     AED 3500
   Selling Price:  AED 8500
   Rental Price:   AED 800/day
   Deposit:        AED 2500

4. Click Save

5. System generates:
   - Product ID: RIAMAN-00001
   - Barcode image
   - Stock level: 0 (add stock next)

6. Add initial stock:
   Go to Inventory → Click [+ Record Movement]
   Movement Type: "Purchase"
   Quantity: 5
   Save

⏱ Time elapsed: 8 minutes

STEP 6: CREATE TEST CUSTOMER (1 minute)
──────────────────────────────────────────────────────────────────

1. Go to Customers sheet

2. Click "[+ Add New Customer]"

3. Enter:
   Name:  Fatima Al Mansoori
   Phone: +971-50-123-4567
   Email: fatima@example.com

4. Click Save

5. System generates Customer ID: CUST-000001

⏱ Time elapsed: 9 minutes

STEP 7: FIRST SALE (1 minute)
──────────────────────────────────────────────────────────────────

1. Go to POS sheet

2. Select Customer: Fatima Al Mansoori

3. Add Item:
   Click [Search Products] → Select "Eternal Dreams"
   Quantity: 1
   Type: Sale

4. Review Total: AED 8,925 (8,500 + 5% VAT)

5. Click [COMPLETE PAYMENT]

6. Invoice generated: INV-20240124-00001
   - Stock automatically decreased (4 remaining)
   - Customer history updated
   - Accounting ledger posted
   - Audit logged

✓ YOUR FIRST SALE IS COMPLETE!

⏱ Time elapsed: 10 minutes

TROUBLESHOOTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

"Macros disabled" message on open?
→ See STEP 1 above. File → Options → Trust Center → Enable macros

Login form doesn't appear?
→ Make sure you imported both VBA files
→ Check _Users_Master sheet exists
→ Press Ctrl+Home, then Ctrl+F to find sheets

Stock numbers wrong?
→ Check _Stock_Movements sheet for all transactions
→ Do physical inventory count
→ Record adjustment movement to correct

Sheets not hiding by role?
→ Clear active workbook
→ Log in again with correct credentials
→ VBA security might need adjustment

Where's my data?
→ Check if you're on correct sheet (not a hidden backend sheet)
→ Use Ctrl+Home to go to top
→ Check sheet tabs at bottom
