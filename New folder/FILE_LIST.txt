✨ KEY FEATURES & INNOVATIONS
══════════════════════════════════════════════════════════════════════════════════

1. MOVEMENT-BASED INVENTORY
   Stock is NEVER manually entered. Every change is a documented movement
   (Purchase, Sale, Rental, Return, Damage, Loss). Complete audit trail.

2. AUTO-GENERATED UNIQUE IDS
   Products: RIAMAN-00001, Customers: CUST-000001, Transactions: INV-YYYYMMDD-XXXXX

3. DYNAMIC AVAILABILITY CHECKING
   Rental system prevents double-booking. Customers see available dates.

4. AUTOMATIC VIP IDENTIFICATION
   Customers automatically marked VIP when spending exceeds threshold.

5. ZERO-TOUCH ACCOUNTING
   Sales automatically post to ledger. No manual journal entries.

6. COMPLETE AUDIT TRAIL
   All actions logged with timestamp, user, details. Compliance-ready.

7. BARCODE SCANNING INTEGRATION
   USB barcode scanners work directly. No configuration needed.

8. ROLE-BASED INTERFACE ADAPTATION
   Different users see different interfaces. Sales see POS, Accountants see reports.

9. MULTI-PAYMENT METHOD SUPPORT
   Cash, Card (POS), Bank Transfer, Online, Stripe, Wise.

10. PROFESSIONAL REPORTING
    Real-time dashboards, daily/monthly reports, profit analysis, tax reporting.

COMMON ISSUES:

"Macros disabled" message?
→ File → Options → Trust Center → Enable macros

Login form not appearing?
→ Check _Users_Master sheet exists
→ Verify both VBA files imported
→ Check Alt+F11 for VBA errors

Stock numbers wrong?
→ Check _Stock_Movements for all transactions
→ Run physical count and adjust via movement

Report numbers don't match?
→ Verify date range
→ Check filter criteria
→ Review _Accounting_Ledger

PASSWORD ISSUES:
→ Change user password in _Users_Master sheet
→ Protect sheets: Right-click sheet → Protect Sheet
→ Unprotect: Right-click sheet → Unprotect Sheet
