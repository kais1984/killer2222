Want to get up and running in 10 minutes? Follow these simple steps:

STEP 1: Enable VBA in Excel (1 min)
├── Open Excel
├── File → Options → Trust Center
├── Trust Center Settings → Macro Settings
└── Select "Enable all macros"

STEP 2: Create Workbook (2 min)
├── Create new blank workbook
├── Press Alt+F11 (open VBA editor)
├── File → Import File → Select "CreateERP_VBA.bas"
└── Press F5 to run the macro

STEP 3: Import Logic (2 min)
├── Press Alt+F11 again
├── File → Import File → Select "RIAMAN_FASHION_ERP_Main.vbs"
└── Close VBA editor (Alt+Q)

STEP 4: Save (1 min)
├── File → Save As
├── Name: RIAMAN_FASHION_ERP.xlsm
├── Type: Excel Macro-Enabled Workbook
└── Save

STEP 5: Add Sample Product (2 min)
├── Go to Products sheet
├── Click [+ Add New Product]
├── Enter: Name, Size, Color, Price, etc.
└── System auto-generates Product ID & Barcode

STEP 6: First Sale (1 min)
├── Go to POS sheet
├── Select or create customer
├── Add product to cart
├── Complete payment
└── Invoice automatically generated!

STEP 7: Explore (1 min)
├── Log in as different users
├── admin / admin123 (see Admin Dashboard)
├── sales / sales123 (see POS)
├── stock / stock123 (see Inventory)
└── accounts / accounts123 (see Reports)

⏱️  Total time: 10 minutes to operational status
