## 🎨 Design System Highlights

### Color Palette
```
Deep Indigo Primary:    #3D4A5D (professional, trustworthy)
Royal Blue Secondary:   #4F8CFF (modern, energetic)
Emerald Success:        #10B981 (confident, positive)
Amber Warning:          #F59E0B (cautious, attention)
Coral Danger:           #EF4444 (clear, necessary)
Off-white Background:   #F6F7F9 (calm, professional)
```

### Typography
- **Font**: Inter (modern SaaS standard)
- **Weights**: 400, 500, 600, 700
- **Scale**: 0.75rem → 2rem
- **Line Height**: 1.6 (readable)

### Spacing (8px grid)
- 4px, 8px, 16px, 24px, 32px, 40px, 48px

### Shadows (5 levels)
- xs (subtle), sm (default), md (hover), lg (modal), xl (dropdown)

### Animations
- Fast (150ms): hover effects
- Base (200ms): card animations  
- Slow (300ms): page transitions

## 🎨 Customization

### Change Primary Color
Edit `static/css/premium-erp.css` line 1:
```css
--primary-700: #your-color;
```
All UI updates automatically!

### Change Font
Edit line 79 of `premium-erp.css`:
```css
--font-inter: 'Your Font', system-ui;
```

### Add Custom Component
See `DESIGN_QUICK_REFERENCE.md` for patterns to copy.
